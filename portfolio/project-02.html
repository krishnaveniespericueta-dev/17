2026 / IP Design

Error
27

Error 27 是《人生正在输入》里的第一位错误码角色，中文名“二七”。它是一只由第 27 次删除的草稿生成的小影子怪，代表一种很日常、也很消耗人的年轻人精神状态：表达失败，请稍后再试。

RoleIP 概念设定与角色设计
Scope角色设定 / UI 语言 / 表情海报 / 错误码矩阵
Status视觉系统与应用展示
Core Sentence 二七不是不会说话。它只是太认真对待每一句话，所以总是发送失败。

它害怕说错话，也害怕让别人失望，所以把很多真心话写进草稿本，最后只回一句“怎么啦？”。这个 IP 的情绪不是单纯的丧，而是“很费劲，但今天也还在运行”。

02

Character
System

二七的识别点由黑灰毛绒、半垂眼、小横嘴、E27 工牌、错误红状态扣、斜挎草稿本和像素故障边缘组成。

03

Interface
Language

系统界面是《人生正在输入》的世界观语法：弹窗、输入框、状态标签和窗口边框把情绪翻译成冷幽默的系统反馈。

04

Emotion
Outputs

第一批传播内容围绕“收到在吗、过度斟酌、假装正常、还没坏彻底”展开，语气保持系统提示与年轻人内耗的反差。

05

Product
Extension

二七适合往毛绒、潮玩、盲盒和桌面摆件延展。产品化重点不是把它做得更夸张，而是保留“安静报错”的气质。

06

Error
Matrix

矩阵页面把《人生正在输入》从单个角色推进到状态宇宙：每个 Error 都代表一种可被理解、可被玩笑化的精神状态。
