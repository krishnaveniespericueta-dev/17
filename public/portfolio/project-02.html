27

Error 27 是《人生正在输入》里的第一位错误码角色，中文名“二七”。它代表一种很日常、也很消耗人的年轻人精神状态：表达失败，请稍后再试。这个 IP 用软萌的影子布偶外形包裹社交内耗、过度斟酌和系统冷幽默。

RoleIP 概念设定与角色设计
Scope世界观 / 主角色 / 道具 / 文案语气
Status第一阶段设定稿

IP
System

“二七”是一只总是跟在别人期待后面的小影子布偶。它害怕说错话，所以把真正想说的话都写进草稿本里，最后只发出一句“怎么啦？”。视觉上保留黑灰毛绒、半垂眼、小横嘴、E27 工牌、斜挎草稿本和像素碎边；语气上保持“系统提示 + 年轻人内耗 + 冷幽默”。后续可以延展为表情包、短漫画、毛绒潮玩，以及 Error 08、Error 404 等错误码角色系列。
