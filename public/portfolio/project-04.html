Reusable Prompt Blueprint

概念锚点 + 主体身份与情绪 + 世界与空间规则 + 材质和光影 + 镜头构图 + 叙事动作 + 负面约束 + 输出检查

每组生成稿先固定角色和视觉语法，再只替换可控变量，例如场景、姿态或画面尺度。这样能在保持系列一致性的前提下，继续拓展海报、设定图、界面应用和关键场景。
